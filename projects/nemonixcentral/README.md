# NemonixCentral — Trading Analysis Platform

## One-Liner
> Financial macro analysis terminal with AWS Bedrock AI. 17+ API endpoints processing real US Treasury data. Built for personal forex trading.

## Why It Matters
This was my first serious full-stack project — a tool I built because I needed it, not because a tutorial told me to. It proves I can ship working software that solves a real problem.

## What It Demonstrates
- Full-stack development (FastAPI + frontend)
- AWS Bedrock integration for AI-powered analysis
- External API integration (US Treasury data)
- API design (17+ REST endpoints)

## Relationship to FTIP
NemonixCentral was the prototype. FTIP is the production redesign.
- NemonixCentral: US-only, monolithic, single AI provider
- FTIP: 8 currencies, microservices, agent architecture, multi-cloud
